Data Analytics Project – README
Project Overview
This project demonstrates a complete Data Analytics workflow using Python, SQL Server, and Power BI. The objective was to load and analyze a dataset, perform Exploratory Data Analysis (EDA), execute SQL queries, build interactive dashboards, and present insights through reports and presentations.
The project highlights data cleaning, analysis, visualization, and business insight generation in a structured and professional manner.

Dataset
The dataset used in this project contains structured business data for analysis purposes. It includes multiple columns related to performance metrics, trends, and operational information.
Key Features of the Dataset

Structured tabular format
Multiple numeric and categorical variables
Suitable for EDA and dashboard creation
Imported and processed using Python and SQL Server

How to Run the Project

Execute provided SQL scripts

Step 5 – Open Power BI Dashboard
